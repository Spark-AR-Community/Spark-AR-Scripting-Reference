# State

### Values <a id="values"></a>

| Value | Description |
| :--- | :--- |
| `AVAILABLE` | Indicates that the resource has been downloaded. |
| `PENDING` | Indicates that the resource download is pending. |
| `FAILED` | Indicates that the resource failed to download. |
