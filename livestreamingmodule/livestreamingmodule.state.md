# LiveStreamingModule.State

### Values <a id="values"></a>

| Value | Description |
| :--- | :--- |
| `LIVE` | Indicates that the effect is streaming live. |
| `NONLIVE` | Indicates that the effect is not streaming live. |
| `PRELIVE` | Indicates that the effect is streaming live. |
