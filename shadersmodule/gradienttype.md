# GradientType

### Values <a id="values"></a>

| Value | Description |
| :--- | :--- |
| `HORIZONTAL` | The gradient will be in horizontal direction. |
| `CIRCULAR` | The gradient will radiate outward in a circular direction. |
| `VERTICAL` | The gradient will be in vertical direction. |
