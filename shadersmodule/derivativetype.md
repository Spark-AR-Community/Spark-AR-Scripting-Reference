# DerivativeType

### Values <a id="values"></a>

| Value | Description |
| :--- | :--- |
| `FWIDTH` | Sum of the absolute derivatives in x and y. |
| `DFDX` | Derivative in x direction. |
| `DFDY` | Derivative in y direction. |
